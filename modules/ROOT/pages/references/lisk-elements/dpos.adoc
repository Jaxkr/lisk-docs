@liskhq/lisk-dpos is DPoS consensus algorithm implementation according to the Lisk protocol.

== Installation

[source,bash]
----
npm install @liskhq/lisk-dpos
----

== Update

To perform an upgrade, execute the following command:

[source,bash]
----
npm update @liskhq/lisk-dpos
----
